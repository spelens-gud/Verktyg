# 产品概述

这是一个 Go 语言基础组件库（go-core），为微服务应用提供标准化的基础设施组件。

## 核心价值

- 提供统一的接口抽象层，实现组件的可替换性
- 集成常用中间件（Redis、MySQL、Kafka、配置中心等）
- 内置可观测性支持（链路追踪、监控指标、日志）
- 简化微服务开发的样板代码

## 主要功能模块

- **配置管理**：支持 ACM、Nacos 等多种配置中心
- **数据存储**：GORM、Redis、Memcached 封装
- **消息队列**：Kafka 生产者/消费者封装
- **HTTP 客户端**：带重试、熔断、追踪的 HTTP 调用库
- **可观测性**：Prometheus 监控、OpenTracing/SkyWalking 追踪、结构化日志
- **服务器框架**：基于 Gin 的 HTTP 服务器，支持优雅重启
- **工具库**：上下文管理、定时任务、并发控制等
